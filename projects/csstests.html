Flexible rows

Useful to position elements side by side and automatically resize them when width changes. In this example, 5 columns become 3 and then 1 in smaller devices. Here's an example of this kind of container. Click the button to add another element to the palette. Resize the page to see how the elements adapt to the width of the page.

To do this we can:

<div class="rowItemContainer">
	<div class="rowItem fifth">1</div>
	<div class="rowItem fifth">2</div>
	<div class="rowItem fifth">3</div>
</div>

The simplest way I found to get this behaviour was using the following CSS:

.rowItemContainer:after, .rowItemContainer:before {
	content: "";
	display: table;
	clear: both;
}

.rowItem {
	float: left;
	box-sizing: border-box;
	padding: var(--default-padding);
}

.fifth {
	width: calc(100%/5);
}

As you can probably imagine, I created very similar code for .quarter and .third so I can have different levels. And then, for the resizing purposes (adding some margin between the sibling elements):

@media only screen and (min-width: 601px) and (max-width: 800px) {
	.fifth {
		width: calc(100%/3);
	}
}

@media only screen and (max-width: 600px) {
	.rowItem {
		width: 100%;
	}

	.rowItem + .rowItem {
		margin-top: var(--default-padding);		
	}
}

HTML to create two columns (becomes 1 in smaller screens).

Remember that the size of these containers can't be manually changed. Don't add margin or padding.

<div class="flexRow">
	<div class="flexRowItem row2">Lorem ipsum.</div>
	<div class="flexRowItem row2">Lorem ipsum.</div>
</div>

The CSS for this is:

.flexRow {
	display: flex;
	justify-content: space-between;
}

.flexRowItem {
	padding: var(--default-padding);
	box-sizing: border-box;
}

.row2 {
	width: calc(50% - var(--default-padding)/2);
	display: inline-block;
}

More columns in each row are also possible, here's an example:

We can do this for any number of columns:

1
2
3
4
5

For this, we create a flexRow container as before and then create <div> of the following classes. The pattern is easy to find, it's something like w = 100%/n - p*(n-1)/n. Check it out:

.row3 {
	width: calc(100% / 3 - var(--default-padding)*2/3);
	display: inline-block;
}

.row4 {
	width: calc(25% - var(--default-padding)*3/4);
	display: inline-block;
}

.row5 {
	width: calc(20% - var(--default-padding)*4/5);
	display: inline-block;
}

Finally, to make it adapt to smaller screens (adding some margin between the sibling elements):

@media only screen and (max-width: 600px) {
	.flexRow {
		display: block;
	}

	.flexRowItem {
		width: calc(100%);
	}
	
	.flexRowItem + .flexRowItem {
		margin-top: var(--default-padding);		
	}
}

Disclaimer: I have no idea if this is the best way to do it. I just wanted to play around with CSS and do it myself, instead of using someone else's stylesheet.
